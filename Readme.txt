IMPORTANT:
* As stated in the rarfile library, it needs a tool to extract a rar file.

TO GET THIS TOOL: //There were a lot of ways to get tools
1. Download winrar.
2. Open winrar file location.
3. Copy the path of UnRar.exe, it can be found inside the winrar file location.
4. Paste the UnRar.exe path inside the "set environmental variables" inside the settings.
For Windows(EASY METHOD for step 3 and 4):
3. Copy the UnRar.exe
4. Paste it in the Windows folder

NOTE:
* Run time will depend on the user input, so read the code/instruction to avoid infinite loop/long waiting time.
